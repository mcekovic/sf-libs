This package contains high-level synchronization primitives.

Classifiers (classes and interfaces) in this package can be divided in three main groups:

Maps and caches with key-level locking:

  • LockableMap and its implementations LockableHashMap, LockableCache, LockableLRUCache, LockableMLFUCache.

    • Execution utilities:

  • NamedThreadFactory
  • FixedRateScheduler

    • High-level synchronization entities:

  • Counter.
